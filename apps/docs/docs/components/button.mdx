import { Button, ThemeProvider } from '@sun/ui';

## 使用示例

<ThemeProvider>
  <div style={{ display: 'flex', gap: 12 }}>
    <Button variant="primary">Primary</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="ghost">Ghost</Button>
    <Button size="lg">Large</Button>
  </div>
  <br />
  <Button fullWidth>Full Width</Button>
  <br />
  <Button disabled>Disabled</Button>
</ThemeProvider>

## 尺寸示例

<ThemeProvider>
  <div style={{ display: 'flex', gap: 12 }}>
    <Button size="sm">Small</Button>
    <Button size="md">Medium</Button>
    <Button size="lg">Large</Button>
  </div>
</ThemeProvider>

## 满宽与禁用

<ThemeProvider>
  <div style={{ width: 320, display: 'grid', gap: 12 }}>
    <Button fullWidth>满宽按钮</Button>
    <Button disabled>禁用按钮</Button>
  </div>
</ThemeProvider>

## 自定义主题颜色

通过 `ThemeProvider` 传入 `theme.colors` 可覆盖主题变量：

```tsx
<ThemeProvider theme={{ colors: { primary: '#10b981' } }}>
  <Button>自定义主题色</Button>
</ThemeProvider>
```

实际效果：

<ThemeProvider theme={{ colors: { primary: '#10b981' } }}>
  <Button>自定义主题色</Button>
</ThemeProvider>

## API 文档

- `variant`: `'primary' | 'outline' | 'ghost'` 默认 `primary`
- `size`: `'sm' | 'md' | 'lg'` 默认 `md`
- `fullWidth`: `boolean` 按钮占满父容器宽度
- 其余属性继承自 `button` 原生属性

## 交互演示

- 更改 `variant` 与 `size` 观察样式变化。
- 在暗黑模式（系统偏好）下自动切换文字与背景色。
