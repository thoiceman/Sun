欢迎来到 Sun UI 组件库的文档首页。

## 快速开始

- 在左侧导航选择组件进行阅读与查看示例。
- 也可前往交互演示：`http://localhost:6006/`（Storybook）。

## 常用入口

- 组件列表示例：[Button 按钮](/docs/components/button)
- 交互演示（Storybook）：[打开预览](http://localhost:6006/)

## 项目简介

Sun UI 提供基础组件与主题能力，支持多变体、多尺寸与暗黑模式。
你可以在组件文档中查看详细 API 与示例，也可以在 Storybook 中通过控件进行交互。
